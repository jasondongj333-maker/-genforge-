# 住宅建筑CAD生成器 - 专注轴网、墙体、门窗的住宅平面图生成

## 用途

生成符合中国建筑 CAD 标准的住宅平面图 C# 代码，覆盖**轴网 → 外墙 → 内墙 → 门窗**完整流程。

## 6阶段绘图流程

严格按以下顺序执行，**禁止跳过阶段**：

| 阶段 | 名称 | 说明 |
|------|------|------|
| 阶段1 | 创建图层系统 | 初始化 A-AXIS / A-WALL / A-DOOR / A-WIND 等标准图层 |
| 阶段2 | 绘制建筑轴网 | 生成定位轴线和轴号（数字+字母） |
| 阶段3 | 绘制外墙 | 外墙厚度 ≥ 240mm，闭合轮廓 |
| 阶段4 | 绘制内墙 | 划分功能空间，厚度 100-200mm |
| 阶段5 | 创建门窗洞口 | 按房间类型自动计算位置，先开洞 |
| 阶段6 | 插入门窗 | 在洞口位置插入门窗线型 |

## 住宅门窗自动布置规则

### 卧室
- **窗**：优先南向，宽度 1500-2100mm，窗台高 900mm
- **门**：单扇平开，宽度 900mm，位置避开床区

### 客厅
- **窗**：优先南向大窗，宽度 2400-3600mm
- **门**：连接入户或过道，宽度 1000mm

### 厨房
- **窗**：优先北向/东向，宽度 1200-1500mm
- **门**：连接餐厅或走廊，宽度 800mm
- **规范**：必须有直接对外通风窗

### 卫生间
- **窗**：北向/东西向高窗，窗台 1500-1800mm
- **门**：宽度 700-800mm，不正对其他房间门
